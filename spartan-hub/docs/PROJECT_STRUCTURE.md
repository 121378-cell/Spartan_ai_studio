# Spartan Hub Project Structure

This document outlines the key file and directory structure of the Spartan Hub project after the Phase 2 consolidation.

## Root Directory (`spartan-hub/`)

*   **`package.json`**: Main configuration file with optimized NPM scripts (replacing legacy .bat/.sh scripts).
*   **`index.html`**: Application entry point, referencing `src/index.tsx`.
*   **`vite.config.ts`**: Vite build configuration.
*   **`docker-compose.yml`**: Docker services definition.
*   **`README.md`**: Project overview.

## Source Code (`src/`)

*   **`index.tsx`**: React application entry point (bootstraps the app).
*   **`main.tsx`**: (Alternative entry point, if present).
*   **`App.tsx`**: Main application component.
*   **`components/`**: React components.
*   **`utils/`**: Utility functions (logger, error handler).
*   **`__tests__/`**: Frontend tests.
    *   `entrypoint.test.js`: Validates `index.html` references.
    *   `scripts.test.js`: Validates `package.json` scripts.

## Backend (`backend/`)

*   **`src/`**: Backend source code.
*   **`dist/`**: Compiled backend code.
*   **`package.json`**: Backend-specific dependencies.

## Documentation (`docs/`)

*   **`INDEX.md`**: Documentation entry point.
*   **`setup/NPM_SCRIPTS.md`**: Reference for new NPM scripts.
*   **`PROJECT_STRUCTURE.md`**: This file.

## Scripts (`scripts/`)

*   **`createDist.js`**: Node.js script for creating distribution packages.
*   **`verifySecrets.js`**: (Planned) Secret verification.
*   **`launcher.js`**: (Planned) Application launcher logic.

## Distribution

*   **`spartan-hub-dist/`**: Generated by `npm run dist`, contains the standalone application package.
