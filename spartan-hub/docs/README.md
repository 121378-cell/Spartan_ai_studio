# Spartan Hub Documentation

This directory contains all project documentation organized by category.

## Directory Structure

```
docs/
├── ai/           # AI service implementations
├── api/          # API and rate limiting docs
├── architecture/ # System architecture and best practices
├── deployment/   # Deployment and Docker setup
├── guides/       # User and developer guides
├── implementation/ # Feature implementation details
├── setup/        # Local development setup
└── logs/         # Application and audit logs
```

## Project Structure

- `src/`: Application source code
- `public/`: Public resources
- `backend/`: Server code
- `docs/`: Documentation (you are here)
- `scripts/`: Utility and test scripts (moved from root)
